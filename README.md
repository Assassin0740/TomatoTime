# 专注计时器 · 终极置顶悬浮版

一款基于 Electron 开发的专注计时器应用，支持任务管理、图片粘贴、悬浮窗等功能。

## ✨ 功能特性

### 🕐 计时功能
- 工作时间/休息时间切换
- 可自定义时间设置
- 倒计时结束提醒（声音+弹窗）
- 暂停/继续功能

### 📝 任务管理
- 支持添加、编辑、删除任务
- 任务优先级设置（高/中/低）
- 任务类型分类（短期/长期）
- 完成状态切换
- 任务评论功能

### 🖼️ 图片支持
- 支持在任务中粘贴图片
- 支持 Ctrl+V 快捷键粘贴
- 支持右键菜单粘贴图片
- 图片自动以 base64 格式保存，重启后正常显示

### 🎨 视图模式
- **卡片视图**：列表式展示任务
- **表格视图**：表格化展示任务
- **纯净模式**：只显示计时器，隐藏任务列表

### 🌊 悬浮窗
- 可打开置顶悬浮窗
- 悬浮窗锁定/解锁功能
- 实时显示倒计时

### ⚙️ 个性化设置
- 背景明暗调节
- 音量大小控制
- 文字大小设置（完整模式/纯净模式）
- 发光效果开关及颜色/强度调节
- 背景图片自定义
- 呼吸效果开关
- 模式标签显示开关

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式运行
```bash
npm start
```

### 打包应用
```bash
npm run build
```

打包完成后，安装包会生成在 `dist` 文件夹中。

## 📖 使用说明

### 基本操作
1. **开始计时**：点击「开始」按钮
2. **暂停计时**：点击「暂停」按钮
3. **切换模式**：点击「切换」按钮（工作 ↔ 休息）
4. **修改时间**：点击分钟或秒钟数字，输入新值

### 添加任务
1. 在输入框中输入任务内容
2. 选择优先级和类型
3. 点击「添加」按钮或按 Enter 键

### 粘贴图片
1. 在任务编辑区域按 Ctrl+V
2. 或右键点击编辑区域，选择「粘贴图片」

### 悬浮窗
1. 点击「悬浮窗」按钮打开
2. 点击锁定按钮锁定/解锁悬浮窗位置

### 视图切换
- 点击「卡片视图」或「表格视图」切换显示方式
- 点击「全部」「短期」「长期」筛选任务类型


## 🔧 配置说明

所有设置均保存在 localStorage 中，包括：
- 计时时间设置
- 背景明暗、音量
- 字体大小、发光效果
- 当前视图模式
- 任务数据

## 📝 快捷键

| 快捷键 | 功能 |
|--------|------|
| Enter | 添加任务（输入框中） |
| Ctrl+V | 粘贴图片 |
